SHSH://How to run a html website.html


1. Open a terminal


2. Cd into your website main dir with the main index.html file in it | EXP: cd /media/user/websites/website-1/


3. Make sure wget is working and installed by running (wget) if it is installed skip this if it is not | Run: sudo apt install wget


4. Run in your terminal | wget https://raw.githubusercontent.com/HttpAnimation/Simple-Html-Sever-Hoster/gh-pages/Sever


5. Make the file runnable | Run: chmod +x Sever


6. Run | ./Sever | If php is not installed it will ask you to install it if you don't need php say no (One time only)
